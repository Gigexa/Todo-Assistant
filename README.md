So this is a todo application with database integration and AI assistant. You will need to have ollama installed in your local device and have mistral model downloaded.
Pretty simple steps takes about 5GB of disk space. 
After that it runs locally on your device and you can have a chat with AI through this app as well as schedule your dotos. Now you can send a voice message to chatbot. It processes English voice to text and prompts mistral, but you will need an openAI apikey for it to use openAI transcription. 
Sorry for the bad UI its not perfect but it works.
